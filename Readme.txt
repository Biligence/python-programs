Querycomputersite:  to check the domain info of given computers. Can be used to identify whether the computer is alive in AD.
                    Input: computer name list in excel format.

SaveDocbypage: Contains two macros
               +Macro SaveAsFileByPage() will save separate one word doc into docs by page and save them as individual docs but without header info.
               +Macro SaveAsFileByPage+Header() will save separate one word doc into docs by page and save them as individual docs but without loosing header info.
